Vite + React + TypeScript with Tailwind CSS preconfigured. Start editing `src/App.tsx` to build out Emoji News.

## Setup

- Install deps: `pnpm install`
- Dev server: `pnpm dev`
- Lint: `pnpm lint`
- Build: `pnpm build`

## Tailwind

- Configured in `tailwind.config.js` with content paths covering `index.html` and `src/**/*.{js,ts,jsx,tsx}`.
- Base styles are in `src/index.css` using `@tailwind base`, `components`, and `utilities`.
- Use the sample UI in `src/App.tsx` as a starting point and keep adding components styled with Tailwind classes.
